Obtaining the documentation of the whole source code of the
C-module find_process and its examples:

1) You need the documentation tool Doxygen.
   If not yet installed, so you can visit the webside of Doxygen:
   http://www.doxygen.org

2) Go in the directory of this readme-file and type:
   doxygen

3) Open in your preferred web-browser the file ./html/index.html
